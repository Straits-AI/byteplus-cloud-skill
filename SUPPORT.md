# Support

This is a community-maintained public-preview project.

- Use a **Bug report** for reproducible problems in the skill or bundled helpers.
- Use a **Capability gap** when a BytePlus workflow is missing, outdated, or only
  partially validated.
- Use a **Feature request** for broader project improvements.
- Use GitHub private vulnerability reporting for security concerns.

Before opening an issue, run the local doctor when safe:

```bash
python3 skills/byteplus-cloud/scripts/byteplus_doctor.py --json
```

Share only sanitized output. Remove account IDs, resource IDs, IP addresses,
request bodies, signed URLs, application data, and all credential-like values.

Questions about BytePlus service availability, billing, quotas, account
entitlements, or product incidents may require official BytePlus support. Project
maintainers cannot change BytePlus account state or service entitlements.
